This is the first robot I built.

designed an autonomous navigation robot on the TurtleBot robot using ROS/C++ to recognize and avoid obstacles while pass though each waypoint.

In this system:

  • Implemented the Extended Kalman filter algorithm for SLAM.
  • Developed a motion planner that was capable of navigating to waypoints in space using A* algorithm based on state lattice.
  • Implemented the measurement models based on the beam model and use laser-scanner sensor from Microsoft Kinect.
